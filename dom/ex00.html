DOM e Formularios

DOM

DOM - Data Object Model é uma API(Application Programming Interface) que visa simplificar a forma de acesar a estrutura de3 um documento HTML. Facilitando criar, modificar, adicionar, retirar e manipular elementos HTML.

Formularios

São elementos HTml que possibilitam a interalção do usario com a página web. Como exemplo temos caixas de texto, botões, caixas de seleção dentre outros. As tags form, label, input são tags de elementos de formularios.

Exemplo

Construa uma interface para qu sejam digitados dois numeros e somados.
