DOM

Metodos

Os metodos permitem manipular elementos do documento HTML com js

  1. document.getElementeByID("nome_do_id") - pega o elemento do HTML via id.
  2. innerHTML - inserir/alterar conteudo.
  3. mudar estilo - alteração de css via js
  4. ClassList - insere uma class no elemento em questão
    • .add - adiciona uma class no elemento
